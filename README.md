# LLMs-generated Policy Expression Dataset

This repository contains a policy expression dataset and long-format outputs generated by large language models (LLMs). The dataset is organized for research use, with policy expressions, model responses, and basic data quality checks provided in both spreadsheet and CSV formats.

## Files

- `cleaned_ai_policy_outputs.xlsx`: Workbook containing the policy expression list, LLM-generated outputs, and data quality checks.
- `policy_expressions_clean.csv`: List of 70 policy expressions with expression IDs, Chinese policy domains, Chinese expressions, and English translations.
- `ai_outputs_long.csv`: Long-format LLM-generated outputs. Each row represents one model-language-task-expression response.
- `data_quality_report.csv`: Summary of basic data checks.

## Structure of `ai_outputs_long.csv`

Each row represents one LLM-generated output and includes fields for:

- model
- expression ID
- policy domain
- Chinese policy expression
- English translation
- output language
- task type
- prompt
- response
- basic response length measures

## Dataset Summary

- Policy expressions: 70

- Expected outputs: 1260
